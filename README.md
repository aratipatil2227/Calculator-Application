## Calculator Application ##
## Project Description##
this is simple calculator application developed using core java.
the application performs basic arithmetic operations such as Addition , Subtraction , Multiplication , Division .

## Features
-user-friendly menu-driven program 
-supports:
-Addition 
-Subtraction
-Multiplication
-Division
-Handles division by zero using exception handling
-uses core java concepts like:
-Methods
-Conditional statements(if-else /switch)
-loops
-Scanner class
-exception handling

## Technologies used
-java
-core java concepts
-cli

## How to run the project

-open the project in any java IDE 
-run the main class
-select opration from the menu
-enter numbers
-view the result
